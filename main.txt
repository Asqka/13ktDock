заметки для себя из задания
COPY app.txt /app/app.txt → копирует файл в контейнер
WORKDIR /app → теперь все команды внутри /app
CMD → что выполняется при запуске контейнера

  cid=$(docker run -d student-ci-app sh -c "sleep 120")
запускаем контейнер в фоне (-d) и сохраняем его ID в переменную cid
 и получается что cid это просто переменная в которую мы посетим  id нашего контейнра и чтобы птом заново все не выводить прост выщзывеам эту переменную
и еще
docker run --rm -e STUDENT_NAME="ASYA" student-ci-app
значит:
создаётся контейнер из образа student-ci-app
внутрь контейнера передаётся переменная:
STUDENT_NAME=ASYA
